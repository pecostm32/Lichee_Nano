This project is made with the Tang Dynasty IDE version "TD_5.0.3_28716_NL"

The project file is awg_v1.al and can be opened with the IDE. To load the bitstream into the FPGA the Anlogic USB-JTAG programmer is needed.
The schematic and firmware for the programmer can be found in here: https://github.com/pecostm32/Lichee_Nano/tree/main/Hardware/Anlogic_JTAG_Programmer
